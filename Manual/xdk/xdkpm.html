11.2 xdkpm

This section is about the Principle Manager.

11.2.1 Invocation

These are the commandline arguments of xdkpm:

  • --help or --nohelp (short version: -h): Display an overview of the commandline arguments. Default: --nohelp.
  • --grammar (-g): Import filter for list of principles from a grammar file. Default: no grammar.
  • --builddeffile or --nobuilddeffile (-b): Build principle definition or directly build principle definition functor and constraints. Default: builddeffile.
  • --ozeditor (-o): Select the editor for *.oz files. Default: "oz".
  • --uleditor (-u): Select the editor for *.ul files. Default: "emacs".
  • --xmleditor (-x): Select the editor for *.xml files. Default: "emacs".
  • --log inspector, --log browser or --log shell (-l inspector, -l browser or -l shell: Show log in Inspector/Browser/Shell. Default: --log inspector.
  • --debug or --nodebug (-d): Switch on the debug mode. Default: nodebug.
